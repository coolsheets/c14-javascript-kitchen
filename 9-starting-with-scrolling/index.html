You may want to scroll your page using javascript.

There are several different scrolling methods including calling "scrollTo" on the element (or documentElement) that you wish to scroll, another method is to target the element you want to come into view and call "scrollIntoView".

A robust approach is to calculate the scroll position you want by using the "offsetTop" property on the element you want to come into view and call the global function "scroll" with that value as the new top.

Add javacsript to the "Scroll" buttons to accomplish scrolling between the two sections of this page.
